# Sales Forecasting Project (Machine Learning | XGBoost + SVR)

This project predicts future sales using advanced machine-learning models (better than LSTM for tabular data).

## Features
- Streamlit Frontend (Upload dataset, Train models, Show metrics, Forecast future sales)
- FastAPI Backend (Provide predictions via API endpoint)
- XGBoost + SVR + RandomForest model comparison
- Automatic “select best model”
- Forecast visualization
